SketchX

SketchX — your ultimate companion for effortless mobile development.
With 5,000+ free icons, ready for both personal and commercial use, creativity has no limits.

Seamless Java & XML integration makes development intuitive, helping Sketchware Pro beginners turn ideas into reality with ease. From design to code, SketchX simplifies your workflow, bridging the gap between inspiration and execution.

*Build smarter. Design faster. Create effortlessly.*


# Features

5000+ SVG Icons (Outline + Filled)
Ready-to-use SVG → Vector XML converter
Smooth integration with Sketchware Pro
Clean asset structure for easy access
Free for personal & commercial use

# How It Works

- Two separate Java class methods handle:
  - SVG parsing
  - Conversion to Vector XML
- Icons are categorized into:
  - "outline"
  - "filled"
- Designed for direct use inside Sketchware Pro projects
